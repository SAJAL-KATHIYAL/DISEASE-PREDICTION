# Diabetes Prediction using Machine Learning

This project predicts whether a person is diabetic or not using a Support Vector Machine (SVM) model.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Steps Performed
1. Data preprocessing
2. Feature scaling using StandardScaler
3. Train-test split
4. Model training using SVM
5. Model evaluation using accuracy score
6. Diabetes prediction system
